# 🧙‍♂️ LifeQuest  
### Turn Your Life Into a Game

LifeQuest is a gamified productivity web application that transforms real-life goals into RPG-style quests. Users earn XP, unlock skill trees, and level up their character by completing habits and milestones.

---

## Overview

LifeQuest reimagines self-improvement as an immersive RPG experience.

Instead of managing traditional task lists, users:

- 🎯 Complete **Main Quests** (long-term goals)
- 🔁 Maintain **Daily Missions** (habits)
- 🌳 Unlock **Skill Trees**
- ⚡ Earn **XP**
- 🆙 Level up their character

The system applies game design principles and behavioral psychology to improve motivation, engagement, and consistency.

---

## Core Features

### 1. Quest System
- Create structured **Main Quests**
- Break them into Sub-Quests
- Track completion percentage
- Earn difficulty-based XP rewards

Example:
Main Quest: Crack Data Science Interview  
Sub-Quests:
- DSA Practice
- ML Revision
- Mock Interviews

---

### 2. Daily Missions (Habit Engine)
- Recurring habit tracking
- Streak multiplier logic
- XP boosts for consistency
- Automatic level progression


---

### 3. XP & Leveling Engine
- Structured XP accumulation
- Level threshold scaling formula
- Achievement badges
- Streak break penalties
- Progress analytics dashboard
