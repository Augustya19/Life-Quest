🧙 LifeQuest
Turn Your Life Into a Game.

A gamified productivity web application that transforms real-life goals into RPG-style quests using XP systems, skill trees, and level progression mechanics.

🚀 Overview

LifeQuest reimagines self-improvement as an immersive RPG experience.

Instead of managing boring task lists, users:

🎯 Complete Main Quests (long-term goals)

🔁 Maintain Daily Missions (habits)

🌳 Unlock Skill Trees

⚡ Earn XP

🆙 Level Up their character

It applies game design principles, behavioral psychology, and progression systems to improve consistency and motivation in real life
